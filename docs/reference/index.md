# Reference

<Collapsible title="To explain the listings" :defaultOpen="true">

In general, the following can be said:

- Portal: helps you move through content
  - navigation layer
- Hub: helps you explore the scene
  - exploration layer
- Index: helps you find specific data
  - lookup layer
- Guide: helps you understand a topic
  - understanding layer
- Archive: helps preserve history for you
  - time/history layer
- Database: helps maintain live structured information for you
  - structured truth layer
- Ranking: helps present information in an objective or subjective manner for you
  - evaluation layer

Layers can reference and feed into each other:
- portal connects you to guides & hubs -> discovery
- hubs connect across all the above -> exploration pathways
- index interacts with the database -> structured lookup
- archive provides historical context to support guides -> credibility
- database can feed rankings -> structured evaluation

With that in mind, many of the entries below may fit into multiple criteria above
